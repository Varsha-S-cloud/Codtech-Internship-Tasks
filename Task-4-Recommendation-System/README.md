# Task 4: Recommendation System

## Dataset
MovieLens 100K dataset containing user ratings for movies.

## Method
Collaborative Filtering using user-user similarity (Cosine Similarity).

## Steps Performed
- Created user-item matrix
- Computed similarity between users
- Generated movie recommendations
- Evaluated model using RMSE

## Tools Used
Python, Pandas, Scikit-learn, Google Colab
